## Keyboard mappings

- Play/Pause: `spacebar`
- Mute/Unmute: `m`
- Fullscreen toggle: `f`
- Volume up/down: `arrow_up` / `awrrow_down`
- Forward/backward: `arrow_right` / `arrow_left`
